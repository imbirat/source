👋 Welcome to Xyrox

Xyrox is a modular, high-performance Discord bot template. This documentation covers the source code structure, command handlers, and integrated database systems.

💡
Pro Tip: All commands are built using Discord's Slash Command API for a better user experience.

Key Architecture

The bot follows a Folder-Based Handler approach. Adding a new command is as simple as creating a file in the /commands directory.

🚀 Installation

To run the Xyrox source code, ensure you have Node.js 18+ installed.

Bash
npm install discord.js @prisma/client dotenv
npm install -D typescript tsx

⚙️ Configuration

Create a .env file in your root directory.

ENV
DISCORD_TOKEN=your_token_here
CLIENT_ID=your_application_id
DATABASE_URL="file:./dev.db"

💰 Economy System

The economy module uses Prisma for high-speed transactions. Balance is cached in-memory to prevent database rate limits.

TypeScript
const balance = await prisma.user.findUnique({ where: { id: userId } });
await prisma.user.update({ where:{id:userId}, data:{coins:newBalance} });

📈 Leveling Module

Features include custom XP rates, level-up announcements, and role rewards.

TypeScript
const xp = calculateXP(message);
await prisma.leveling.update({ where:{id:userId}, data:{xp:xp} });

💣 Anti-Nuke Logic

Prevents unauthorized mass deletions. If a user deletes a channel, the bot recreates it instantly.

TypeScript
guild.channels.create({ name: channel.name, type: channel.type });

🛡️ AutoMod

Integrated filters for links, caps, and spamming using a regex-based detection system.

⌨️ Commands List

⚔️ Moderation

  • /kick [user] [reason] Kick a member from the server.
  • /ban [user] [reason] Ban a member from the server.
  • /clear [amount] Delete 1-100 messages from a channel.

💰 Economy

  • /cash [user] Check cash balance.
  • /daily Claim daily reward.
